# Документация KDS (Kitchen Display System)

## Обзор

Система управления кухонными дисплеями для ресторанов с поддержкой нескольких филиалов.

## Структура проекта

```
src/
├── main/
│   ├── java/
│   │   └── ru/sushi/delivery/kds/
│   │       ├── service/
│   │       │   ├── MultiCityViewService.java      # Чтение данных городов
│   │       │   └── MultiCityOrderService.java     # Работа с заказами
│   │       ├── view/
│   │       │   └── CreateNewOrderView.java        # UI создания заказов
│   │       └── config/
│   │           └── MultiCityDataSourceConfig.java # Конфигурация БД
│   └── resources/
│       └── application.properties                 # Настройки подключений
└── test/
    └── java/
```
